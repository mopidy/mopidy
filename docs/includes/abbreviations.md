*[backend]: A Mopidy component providing music library, playlist storage, and/or playback from a music source or service.
*[core]: The central Mopidy component that coordinates multiple frontends and backends, and owns the tracklist.
*[extension]: A Python package that adds one or more backends, frontends, mixers, or web clients to Mopidy.
*[frontend]: A Mopidy component that uses the core API, such as an MPD server, HTTP server, or scrobbler.
*[mixer]: A Mopidy component that controls audio volume and muting.
*[tracklist]: Mopidy's name for the play queue; the ordered list of tracks to be played.
