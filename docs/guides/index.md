
# Guides

Step-by-step guides for specific use cases beyond the basic setup.

## Hardware

- [Raspberry Pi](raspberrypi.md) — set up Raspberry Pi OS and Mopidy on a
  Raspberry Pi.

## Audio output

- [Audio sinks](audiosinks.md) — configure GStreamer audio output for
  different hardware and software audio systems.
- [Icecast](icecast.md) — stream audio from Mopidy through an Icecast server
  so multiple clients can listen simultaneously.
- [PulseAudio](pulseaudio.md) — connect Mopidy running as a system service to
  a PulseAudio server running as your main user.
- [PipeWire](pipewire.md) — connect Mopidy running as a system service to
  a PipeWire server running as your main user.

## Remote control

- [UPnP](upnp.md) — make Mopidy available as a UPnP/DLNA media renderer or
  expose its music library to UPnP media players on your network.

## Development

- [Extension development](extensiondev.md) — build your own Mopidy extension
  to add new music sources, frontends, or other functionality.
