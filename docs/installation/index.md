# Installation

- [Install from system packages](packages.md) — install from your operating system's package
  manager for Debian/Ubuntu, Arch Linux, Fedora, or macOS.
- [Manual install](manual.md) — install from PyPI using `pip`, for systems
  not covered above.

If you want to contribute to Mopidy, install it the regular way first, then
read [Contributing](../about/contributing.md) and
[Development environment](../about/devenv.md).
