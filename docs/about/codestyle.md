
# Code style

All projects in the Mopidy organization use
[Ruff](https://github.com/astral-sh/ruff) for code style enforcement:

- Automatically format all code, using the default configuration.
- Automatically sort imports, using the default configuration.
- As far as reasonable and possible, comply with the lint warnings.

Strict adherence is enforced by our CI setup.
Pull requests that do not pass these checks will not be merged.
