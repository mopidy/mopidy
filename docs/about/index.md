
# About

Mopidy is an open source project. Here is where to find us.

## Getting help

The best place to ask for help is the [Discourse forum](https://discourse.mopidy.com/).
You can also find users in the `#mopidy-users` stream on
[Zulip chat](https://mopidy.zulipchat.com/).

If you've found a bug or have a feature request, please create an issue in the
[issue tracker](https://github.com/mopidy/mopidy/issues). If you're unsure
whether it's a bug or not, ask in the forum first.

## Getting involved

The best place to follow and participate in Mopidy development is the `#mopidy-dev`
stream on [Zulip chat](https://mopidy.zulipchat.com/). Announcements also appear
on the [Discourse forum](https://discourse.mopidy.com/).

## Source code

Mopidy's source code is on [GitHub](https://github.com/mopidy/mopidy), along
with many of the extensions maintained by the Mopidy organization.

Mopidy has been in development since 2009, built by [many contributors](authors.md)
and supported by [sponsors](sponsors.md).
