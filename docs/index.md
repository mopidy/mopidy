# Mopidy

[Mopidy](https://mopidy.com/) is an extensible music server written in Python.

These are the user and developer docs. For the project pitch, screenshots, and
news, see [mopidy.com](https://mopidy.com/).

## Getting started

- [Installation](installation/index.md) — install Mopidy on Linux, macOS, or
  Raspberry Pi.
- [Usage](usage/index.md) — run the server, configure it, and connect a
  client.
- [Extensions](ext/index.md) — add music sources, frontends, and more.

## Going further

- [Guides](guides/index.md) — setup walkthroughs, audio output, and extension
  development.
- [Reference](reference/index.md) — command line, config, APIs, and glossary.
- [About](about/index.md) — contributing, changelog, authors, and sponsors.

## Getting help

If you get stuck, ask for help on the [Discourse
forum](https://discourse.mopidy.com/).

If you stumble into a bug or have a feature request, please create an issue in
the [issue tracker](https://github.com/mopidy/mopidy/issues). If you're unsure
if it's a bug or not, ask on the forum first. The [source
code](https://github.com/mopidy/mopidy) may also be of help.

## Getting involved

To follow and discuss Mopidy development, see the [Development
category](https://discourse.mopidy.com/c/dev/10) on the Discourse forum, or
the `#mopidy-dev` stream on [Zulip chat](https://mopidy.zulipchat.com/).
